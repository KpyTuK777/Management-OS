# Management OS

## Architecture Principles

### 1. Home відповідає лише за сьогодні.

### 2. Кожна інформація має своє законне місце.

### 3. Один модуль — одна відповідальність.

### 4. Якщо модуль займає більше одного екрана — він переходить на окрему сторінку.

### 5. Інтерфейс — українською.

### 6. Професійні терміни — англійською.

### 7. Всі відступи кратні 8px.

### 8. Спочатку система, потім функціональність.
