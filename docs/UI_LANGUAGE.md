# UI Language

This legacy documentation path is retained for compatibility.

The canonical standard for Ukrainian interface terminology, writing style,
component naming, and future AI communication is the
[Product Language Guide](PRODUCT_LANGUAGE.md).

Operational Situation Board naming and reasoning-change language are governed by
the Product Language Guide and the
[Operational Situation Board Architecture](OPERATIONAL_SITUATION_BOARD_ARCHITECTURE.md);
this compatibility path does not redefine them.
