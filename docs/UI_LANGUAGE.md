# UI Language

## Rules

- Product UI text is Ukrainian.
- Use concise, action-oriented labels.
- Keep a term consistent across navigation, headings, buttons, empty states, and messages.
- Capitalize only the first word in Ukrainian headings and button labels unless a proper name or acronym requires otherwise.

## Dates

- Display dates in the Ukrainian locale.
- Use `uk-UA` formatting in JavaScript for user-facing dates.
- Prefer a localized date over an ambiguous numeric format when space allows.

## Professional English terms kept intentionally

The following terms may remain English because they are established professional labels: `KPI`, `SOP`, `Dashboard`, `Deep Work`, and `Global Search`.
