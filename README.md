# Management OS

Management OS is a personal operating-system prototype for an operations manager. It brings projects, daily focus, knowledge, reviews, and operational planning into one interface.

## Goals

- Keep the current day focused and actionable.
- Maintain a clear portfolio of projects.
- Build a durable knowledge and process system over time.
- Reduce operational overhead and support better decisions.

## Current status

The project is an early static prototype. The Portfolio module supports creating, editing, deleting, rendering, and persisting projects in the browser. The dashboard is currently static. Knowledge Base, Review Center, Analytics, Calendar, Tasks, and Settings are planned or placeholder systems.

## Technology stack

- HTML
- CSS
- Vanilla JavaScript
- Browser `localStorage`

## Run locally

Open the repository in VS Code and launch `index.html` or `portfolio.html` with the Live Server extension. No installation, build step, or backend is required.
